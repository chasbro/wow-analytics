== Description ==

Embeds the WOW Analytics tracking code into your Wordpress website without having to edit your templates.

== Installation ==

1. Upload WOW Analytics to the /wp-content/plugins/ directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the WOW Analytics panel under the 'Settings' menu and add your Client Id and set the settings you want.
